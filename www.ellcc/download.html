The ELLCC project is supported by Pennington Software

Tar Balls

Pre-built binaries and target libraries are available for download. You can get a feel for what's in the tarballs by looking at the directory tree.

The distribution includes ecc, the C/C++ compiler, assemblers for all target processors, a linker, debugger, and other utilities that support all the targets, and qemu for each target. The distribution also includes the source of the entire run-time library, and compiled libraries for all the targets.

You can find out more about what's in the release by looking at the ChangeLog and the Bug Database.

Unpacking

You can untar the distribution anywhere:

[~] main% tar xvfpz ~/Downloads/ellcc-0.0.0-linux-x86_64.tgz

An Example

In this example we compile a simple "hello world" program and run it native and for the Mips.

[~] main% cd ellcc-0.0.0
[~/ellcc-0.0.0] main% cat hello.c
#include <stdio.h>

int main(int argc, char** argv)
{
    printf("hello world\n");
}
[~/ellcc-0.0.0] main% bin/ecc hello.c
[~/ellcc-0.0.0] main% ./a.out
hello world
[~/ellcc-0.0.0] main% bin/ecc -target mips-ellcc-linux hello.c
[~/ellcc-0.0.0] main% bin/qemu-mips a.out
hello world
[~/ellcc-0.0.0] main%
