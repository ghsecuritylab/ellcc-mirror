 Pennington Software supports development of the ELLCC project.

What is ELLCC?

ELLCC is a tool chain specifically targeting embedded platforms.
  • C/C++ compiler
  • Sophisticated whole program optimization and code generation
  • GNU binutils
  • GNU GDB source level debugger
  • Multiple target support: ARM, Microblaze, Mips, Nios2, PowerPC, and X86
  • Support of stand-alone and operating system based systems
